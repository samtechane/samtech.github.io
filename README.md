# Ethiopian Orthodox Audio Library

This project is a free web platform created to provide Ethiopian Orthodox Church members with easy access to spiritual audio recordings, including teachings, sermons, and chants related to annual church celebrations.

The recordings are organized according to the Ethiopian calendar months, allowing users to follow the church year in a structured and culturally meaningful way.

## Features
- Audio playback (play, pause)
- Repeat functionality
- Playback speed control (up to 2×)
- Forward and rewind
- Audio download for offline listening
- Organized by Ethiopian calendar months
- Simple and accessible design for all age groups

## Purpose
The goal of this project is to preserve, share, and support Ethiopian Orthodox traditions through modern technology while respecting the spiritual and cultural importance of the content.

## Technology Used
- HTML
- CSS
- JavaScript
- GitHub Pages (free hosting)

## Disclaimer
All audio recordings are shared for spiritual and educational purposes only.  
Rights to the recordings remain with their respective churches, speakers, and contributors.
